Privacy Policy

R13L only collects server IDs and very limited user data, strictly for functionality across the following categories. Users may opt in or out where applicable:

  • Moderation system: Stores warning expiration data to track moderation actions.
  • Economy category: Stores user IDs to track virtual currency balances.
  • Welcome category: Stores channel IDs for welcome messages and role IDs for auto-assigned roles.
  • Starboard category: Stores message IDs only (never message content) to display messages in the starboard.

No message content, personal data, or other identifiable information is collected by R13L.

All data is used solely for in-bot functionality and is never shared with third parties.

Data is stored securely and can be deleted upon request.

By using R13L, you agree to this privacy policy.
